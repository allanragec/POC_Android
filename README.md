# POC_Android

Just a POC for a client using some features of Liferay SDKs

Using:

Technologies/SDKs
  - Kotlin
  - RxAndroid 2
  - Retrofit 2
  - Retrofit 2 with RxJava2 adapter
  - Android Disposebag

Liferay SDKs
  - Liferay Mobile SDK
  - Liferay Push
  - Liferay Screenlets (Login and WebContentDisplay)
